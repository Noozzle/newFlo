  Нормалізована схема

  ┌─────────────────┬───────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┬─────────────────────────────────────┐
  │     Таблиця     │                                                                        Колонки                                                                        │       Відсутні (unavailable)        │
  ├─────────────────┼───────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┼─────────────────────────────────────┤
  │ candles_1m      │ ts (ms int), symbol, open, high, low, close, volume                                                                                                   │ turnover, trades_count              │
  ├─────────────────┼───────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┼─────────────────────────────────────┤
  │ candles_15m     │ те саме                                                                                                                                               │ те саме                             │
  ├─────────────────┼───────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┼─────────────────────────────────────┤
  │ l1              │ ts (ms int), symbol, bid_price, bid_size, ask_price, ask_size, mid_price, spread                                                                      │ bid_depth_5, ask_depth_5 (немає L2) │
  ├─────────────────┼───────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┼─────────────────────────────────────┤
  │ exchange_trades │ ts (ms int), symbol, price, amount, side                                                                                                              │ trade_id, is_block_trade            │
  ├─────────────────┼───────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┼─────────────────────────────────────┤
  │ my_trades       │ ts_entry (ms), ts_exit (ms), trade_id, symbol, side, entry_price, exit_price, size, gross_pnl, fees, slippage_est, net_pnl, exit_reason, hold_seconds │ signal_metadata, risk_amount, net_R │
  └─────────────────┴───────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────┴─────────────────────────────────────┘

  Timestamp нормалізація: всі CSV мають ISO datetime різної точності. Для ML pipeline краще конвертувати в int64 ms since epoch — уніфікує join і сортування. spread = ask_price - bid_price (derived, додається при read).
